# ldap_bofs

BOFs I needed when proxying sucked or wasn't viable

## Installation

Requirements

```
uv
```

Install uv and the dependencies to use the helper python scripts in each directory

```
git clone https://github.com/garrettfoster13/ldap_bofs/
cd project-name
uv sync
```
